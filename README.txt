# Loan Prediction Project

This project analyzes loan application data to predict loan approvals using machine learning models.

## Contents
- `loan_predictions_project.ipynb`: Jupyter notebook with analysis and models
- `data/`: dataset used
- `requirements.txt`: Python dependencies

## Models Tried
- Logistic Regression
- Random Forest
- XGBoost

## How to Run
1. Clone this repo
2. Install requirements: `pip install -r requirements.txt`
3. Open Jupyter Notebook and run the notebook
